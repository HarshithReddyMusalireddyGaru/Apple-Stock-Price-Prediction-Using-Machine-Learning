# Apple-Stock-Price-Prediction-Using-Machine-Learning
This project focuses on predicting the stock price of Apple Inc. (AAPL) using historical stock market data. The goal is to analyze past trends, perform data preprocessing, and apply machine learning or time series techniques to forecast future stock prices.
 Key Components

Data Collection (Historical AAPL stock data)

Data Cleaning & Handling Missing Values

Exploratory Data Analysis (EDA)

Correlation Analysis

Moving Averages (Short-term & Long-term trends)

Model Building (e.g., Linear Regression, LSTM, ARIMA)

Model Evaluation

 Technologies Used

Python

Pandas

NumPy

Matplotlib / Seaborn

Scikit-learn

TensorFlow / Keras (if LSTM used)
